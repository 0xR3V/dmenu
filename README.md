# Installation

## Dependencies
- `noto-fonts-emoji`.
- `libxft-bgra` from the [aur](https://aur.archlinux.org/packages/libxft-bgra).

After installing the dependencies, install my build of dmenu:

```sh
git clone https://github.com/0xR3V/dmenu.git
cd dmenu
sudo make clean install
```
